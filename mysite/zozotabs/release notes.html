Release notes for Zozo Tabs v.2.2

Release date: January 22 2013

Tabs

Features

  • Fully documented and step-by-step guide provided to help you get up and running with Zozo Tabs. (www.zozoui.com)
  • Slick design
  • Limitless customization
  • Horizontal & Vertical Tabs
  • 10 Examples included so that you can see how to implement these in your own website or project
  • 10 Flexible ways to position (top-left, top-center, top-right, top-compact, bottom-left, bottom-center, bottom-right, bottom-compact)
  • 10 built-in themes that are ready to go (white, crystal, silver, gray, black, orange, red, green, blue, deepblue)
  • All tabs comes in 6 different sizes (mini, small, medium, large, xlarge, xxlarge)
  • Multiple Instances: multiple tabs allowed on one page and can have different themes and options
  • Nested Tabs: multiple and nested tabs can different themes and options with any conflict
  • Underlined Tabs
  • Autoplay support with optional stop on hover/click
  • CSS3 & HTML5 design
  • SEO Friendly
  • Load from Ajax (Dynamically add & remove tabs and it's content)
  • Bookmarkable tabs
  • Animation: adjustable speed and effects (fadeIn/slideDown)
  • Codeless creation & customization with HTML5 data attributes
  • Ability to change tabs event (click/mousehover)
  • Event on selecting a tab (callbacks)
  • Public methods (select, first, last, next and previous, etc.)
  • Compatible with all major browsers (including IE7, IE8, IE9 , Chrome, Firefox, Safari and Opera)
  • Free support and updates
  • Extended HTML documentation
